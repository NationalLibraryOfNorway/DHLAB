## Installation

Ensure that you already have  [Python](https://www.python.org/downloads/) >=3.10 installed.

Install the latest version of the [`dhlab`](https://pypi.org/project/dhlab/) python package in your (Unix) terminal with pip:

```shell
pip install -U dhlab
```
